# Antigravity Simulation

## Как запустить проект

### 1. Предварительная настройка
Убедитесь, что у вас есть Python 3 и установлен пакет `pybind11`:
```bash
./venv/bin/pip install pybind11
```

### 2. Компиляция C++ ядра
Для ускорения расчетов используется модуль на C++. Его нужно скомпилировать перед первым запуском или после любых изменений в папке `core/src/`.

Запустите скрипт сборки:
```bash
./build.sh
```

### 3. Запуск симуляции
Чтобы открыть графический интерфейс:
```bash
./run.sh
```

---

## Структура проекта
- `main.py`: Точка входа.
- `model/`: Логика агентов (Python).
- `core/src/`: Высокопроизводительное ядро (C++).
- `gui/`: Графический интерфейс.
- `logs/`: Логи симуляции.
